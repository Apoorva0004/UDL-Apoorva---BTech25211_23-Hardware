# Astable Multivibrator PCB

This project contains the schematic and PCB design of an Astable Multivibrator created using KiCad.

## Features
- Dual transistor astable multivibrator
- Through-hole components
- Single layer PCB routing
- Gerber and drill files generated

## Design Details
- Track width used: 0.5 mm
- Silkscreen labels added
- Mounting holes included

## Files Included
- KiCad source files
- Gerber files
- Drill files
- BOM CSV
- Interactive BOM HTML
- Schematic PDF

## Resources Used
- KiCad documentation
- PCB routing tutorials
- BC547 datasheet
